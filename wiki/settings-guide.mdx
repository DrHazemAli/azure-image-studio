## ‚òÅÔ∏è Azure Configuration

### Basic Azure Setup

Navigate to **Settings** ‚Üí **Azure** to configure your Azure AI services.

#### Required Information

- **Azure Endpoint**: Your Azure AI resource endpoint
- **API Key**: Primary or secondary API key
- **Region**: Azure region (e.g., East US, West Europe)

#### Advanced Azure Settings

- **Multiple Endpoints**: Add backup endpoints for reliability
- **Custom Headers**: Add authentication headers
- **Retry Logic**: Configure retry attempts and delays
- **Rate Limiting**: Set request rate limits

### Model Configuration

#### Adding Models

1. Go to **Settings** ‚Üí **Models**
2. Click **"Add Model"**
3. Select model type:
   - **DALL-E 3**: Image generation
   - **FLUX 1.1 Pro**: Advanced creative models
   - **Florence 2.0**: Vision-language model
   - **GPT-Image-1**: Multimodal capabilities

#### Model Settings

- **Model Name**: Custom display name
- **API Endpoint**: Specific model endpoint
- **Parameters**: Custom generation parameters
- **Cost Tracking**: Monitor usage and costs

## üíæ Settings Management

### Export/Import Settings

1. Go to **Settings** ‚Üí **General**
2. Click **"Export Settings"**
3. Choose what to export:
   - All settings
   - Azure configuration only
   - UI preferences only
   - Custom selections

### Reset Settings

1. Go to **Settings** ‚Üí **General**
2. Click **"Reset Settings"**
3. Choose reset type:
   - **Soft Reset**: Keep user data
   - **Hard Reset**: Complete reset
   - **Selective Reset**: Reset specific categories

### Settings Backup

- **Automatic Backup**: Daily settings backup
- **Manual Backup**: On-demand backup
- **Cloud Backup**: Sync settings to cloud
- **Version History**: Track settings changes
