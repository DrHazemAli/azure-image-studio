## Studio Interface Overview

### Mac-Style Window Controls

AI Image Studio features a modern macOS-inspired interface with:

- **Traffic Light Controls:** Red, yellow, and green window controls for close, minimize, and maximize
- **Unified Title Bar:** Shows application name and editable project name
- **Blurred Glass Effect:** Modern backdrop-blur appearance throughout the interface

#### Project Name Management

- **Click-to-Edit:** Click on any project name to rename it instantly
- **Auto-Save:** Project names are automatically saved when changed

## AI Generation Features

### Text-to-Image Generation

#### Supported Models

##### DALL-E 3 (Azure OpenAI)

- **Strengths:** Reliable, fast generation
- **Best For:** General purpose, photorealistic images
- **Limitations:** Single image output
- **Quality:** Standard and HD options

##### FLUX 1.1 Pro (Black Forest Labs)

- **Strengths:** Artistic quality, style consistency
- **Best For:** Creative and artistic content
- **Features:** Advanced style control
- **Quality:** High resolution output

##### FLUX Kontext Pro (Black Forest Labs)

- **Strengths:** Context awareness, editing capabilities
- **Best For:** Complex scenes, object relationships
- **Features:** Multi-object generation, spatial reasoning
- **Quality:** Professional-grade output

##### GPT-Image-1 (Premium Azure OpenAI)

- **Strengths:** Latest AI technology, premium features
- **Best For:** Commercial and professional use
- **Features:** Advanced quality settings, priority processing
- **Quality:** Highest available quality

#### Generation Parameters

##### Core Parameters

```javascript
const generationConfig = {
  prompt: "Detailed description of desired image",
  model: "dalle-3", // or flux-1-1-pro, flux-kontext-pro, gpt-image-1
  size: "1024x1024", // or 1792x1024, 1024x1792
  quality: "hd", // standard, hd, high (premium)
  style: "vivid", // natural, vivid, artistic
  count: 1, // 1-4 images
  seed: 123456, // for reproducibility
  negativePrompt: "blurry, low quality", // what to avoid
};
```

##### Advanced Parameters

- **Guidance Scale:** How closely to follow prompt (1-20)
- **Steps:** Number of generation steps (10-50)
- **Sampler:** Algorithm type (DPM++, Euler, etc.)
- **CFG Scale:** Creative freedom vs. prompt adherence
- **Clip Skip:** Skip layers for different styles

#### Prompt Engineering

##### Effective Prompt Structure

1. **Subject:** Main focus of the image
2. **Style:** Artistic style or medium
3. **Composition:** Layout and framing
4. **Lighting:** Lighting conditions
5. **Colors:** Color palette preferences
6. **Quality:** Quality modifiers

##### Example Prompts

```
Basic Prompt:
"A red rose in a vase"

Enhanced Prompt:
"A single red rose in an elegant crystal vase, soft natural lighting from a window, shallow depth of field, photorealistic style, high detail, professional photography"

Artistic Prompt:
"A red rose painted in the style of Van Gogh, swirling brushstrokes, vibrant colors, impasto technique, oil painting on canvas, dramatic lighting, museum quality"
```

##### Prompt Modifiers

- **Quality:** "4K", "8K", "high resolution", "professional"
- **Style:** "photorealistic", "oil painting", "digital art", "watercolor"
- **Lighting:** "golden hour", "studio lighting", "dramatic shadows"
- **Camera:** "wide angle", "macro lens", "telephoto", "fish-eye"
- **Composition:** "rule of thirds", "symmetrical", "close-up", "panoramic"

## Troubleshooting and Support

### Common Issues and Solutions

#### Generation Problems

##### Failed Generations

**Issue:** AI generation fails with error message
**Solutions:**

1. Check internet connection
2. Verify API keys in settings
3. Try different model
4. Reduce prompt complexity
5. Check API quota limits

**Issue:** Generated images are poor quality
**Solutions:**

1. Use more detailed prompts
2. Select higher quality settings
3. Try different AI model
4. Adjust generation parameters
5. Use negative prompts to exclude unwanted elements

##### Performance Issues

**Issue:** Slow generation times
**Solutions:**

1. Check internet speed
2. Try different time of day
3. Switch to faster model
4. Reduce image resolution
5. Clear browser cache

#### Interface Problems

##### Panel and Layout Issues

**Issue:** Panels not displaying correctly
**Solutions:**

1. Refresh browser
2. Clear browser cache
3. Reset panel layout in settings
4. Check browser zoom level
5. Try different browser

##### Keyboard Shortcuts Not Working

**Issue:** Shortcuts don't respond
**Solutions:**

1. Check focus on canvas/app
2. Verify shortcuts in settings
3. Check for browser conflicts
4. Try clicking on canvas first
5. Reset shortcuts to defaults
