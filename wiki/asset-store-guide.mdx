## üéØ Overview

The Asset Store is your gateway to millions of high-quality images, graphics, and design elements. It seamlessly integrates with leading stock photo services and provides powerful tools for organizing, searching, and managing your creative assets.

## üè™ Supported Providers

### Pexels Integration

- **Commercial Use**: All images are free for commercial use
- **API Integration**: Direct API access for seamless browsing
- **High Resolution**: Images up to 4K resolution

### Unsplash Integration
- **Premium Quality**: Curated collection of beautiful photos
- **Professional Grade**: High-quality images for professional use
- **API Access**: Full API integration with search capabilities
- **Attribution**: Automatic attribution handling

### Local Asset Library
- **Personal Collection**: Your own uploaded assets
- **Organization**: Folders, tags, and categories
- **Search**: Full-text search across your library
- **Sync**: Cloud sync for multi-device access

## üöÄ Getting Started

### Accessing the Asset Store
1. **Open Studio**: Launch Azure Image Studio
2. **Asset Store Tab**: Click the "Asset Store" tab in the left sidebar
3. **Provider Selection**: Choose between Pexels, Unsplash, or Local Library
4. **Start Browsing**: Begin exploring available assets

### First-Time Setup
1. **API Keys**: Configure API keys in Settings ‚Üí Asset Store
2. **Preferences**: Set your default search preferences
3. **Download Location**: Choose where to save downloaded assets
4. **Sync Settings**: Configure cloud sync if desired

## üîç Search and Discovery

### Basic Search
1. **Search Bar**: Enter keywords in the search box
2. **Filters**: Use filters to narrow results:
   - **Orientation**: Portrait, landscape, square
   - **Size**: Small, medium, large, extra large
   - **Color**: Dominant color filter
   - **Category**: Nature, people, business, etc.
3. **Sort Options**: Sort by relevance, date, popularity
4. **Results**: Browse through search results

### Advanced Search
1. **Boolean Operators**: Use AND, OR, NOT in searches
2. **Phrase Search**: Use quotes for exact phrases
3. **Wildcards**: Use * for partial matches
4. **Exclusions**: Use minus (-) to exclude terms
5. **Color Search**: Search by specific colors

### Search Examples
```
# Basic searches
"mountain landscape"
sunset AND ocean
business people NOT formal

# Advanced searches
"coffee shop" AND "morning light"
nature -animals
"*technology*" AND modern
```

## üìÅ Asset Management

### Adding Assets to Projects
1. **Browse Assets**: Find the asset you want
2. **Preview**: Click to see full-size preview
3. **Add to Project**: Click "Add to Project" button
4. **Layer Creation**: Asset automatically becomes a new layer
5. **Positioning**: Drag to position on canvas

### Downloading Assets
1. **Select Asset**: Click on desired asset
2. **Download Options**: Choose resolution and format
3. **Download Location**: Select save location
4. **Metadata**: Include attribution and metadata
5. **Download**: Click "Download" to save locally

### Local Library Management
1. **Upload Assets**: Drag and drop files to upload
2. **Organize Folders**: Create custom folder structure
3. **Add Tags**: Tag assets for easy searching
4. **Edit Metadata**: Add descriptions and keywords
5. **Sync Settings**: Configure cloud synchronization

## ‚öôÔ∏è Settings and Configuration

### Provider Settings
Navigate to **Settings** ‚Üí **Asset Store** to configure:

#### Pexels Configuration
- **API Key**: Your Pexels API key
- **Search Quality**: Image quality preference

#### Unsplash Configuration

- **Usage Tracking**: Monitor API usage and limits

### Download Settings
- **Default Format**: PNG, JPG, or WebP
- **Resolution**: Choose default resolution
- **Compression**: Quality vs. file size balance
- **Metadata**: Include/exclude metadata
- **Attribution**: Automatic attribution handling

### Cache Management
- **Cache Size**: Set maximum cache size
- **Cache Location**: Choose cache directory
- **Auto-Cleanup**: Automatic cache cleanup
- **Offline Access**: Enable offline asset access
